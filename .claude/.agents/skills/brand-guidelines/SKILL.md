# Brand Guidelines

## Colors (customize in globals.css)
- Primary: `hsl(191, 82%, 30%)` — teal
- Background: `#FFFFFF` / `#1A1A1A`
- Muted: `#F5F5F0` warm beige
- Border: `#E5E5E0`

## Typography
- Headings: Bricolage Grotesque (change in layout.tsx)
- Body: Geist Sans
- Code: Geist Mono

## Customizing
1. Edit CSS variables in `app/globals.css`
2. Change heading font in `app/layout.tsx`
3. Replace `/public/favicon.svg`
