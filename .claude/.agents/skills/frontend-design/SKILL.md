This skill guides creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics.

## Design Thinking

Before coding, commit to a BOLD aesthetic direction:
- **Purpose**: What problem does this interface solve?
- **Tone**: Brutally minimal, maximalist, retro-futuristic, luxury, playful, editorial, brutalist, etc.
- **Differentiation**: What makes this UNFORGETTABLE?

## Guidelines

- **Typography**: Distinctive font choices. Avoid Inter, Roboto, Arial.
- **Color**: Dominant colors with sharp accents. Use CSS variables.
- **Motion**: High-impact moments — page load reveals, scroll triggers, hover states.
- **Layout**: Unexpected layouts. Asymmetry. Grid-breaking. Generous negative space.
- **Depth**: Gradient meshes, noise textures, geometric patterns, layered transparencies.

NEVER use generic AI aesthetics. Choose a direction and execute with precision.
